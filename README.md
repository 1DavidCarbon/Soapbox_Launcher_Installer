## Included in Installer Version
- .NET Framework 4.6.1 & 4.6.2 (Web Installer)
- Official Game Launcher
- Tools
  - Vinyl Manager
- [x] Custom Directory

## Requirement to install Game Launcher
- [X] .NET Framework [4.6.1 on Windows 8] / [4.6.2 (Minimum)]
- [X] Windows 7 SP1 or Newer
- [x] Admin Rights
